## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Emauri-Neto/FoodHarbor-FrontEnd
```

Entre no diretório do projeto

```bash
  cd foodharbor-frontend
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
